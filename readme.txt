Adds a text captcha to Contact Form 7

== Description ==

Add an text captcha to your contact form 7 form. This plugin will not slow down your site with additional header requests like Google's ReCAPTCHA and respects your users privacy.

= Directions [PLEASE READ] =
Add the shortcode [cf7tc] to the form editor where you want the captcha to appear.

== Installation ==

1. Upload contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[cf7tc]` to your contact form 7 forms

== Frequently Asked Questions ==

= How do you add the Text CAPTCHA to the forms? =

Simply add this shortcode `[cf7tc]` to your contact form
